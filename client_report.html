Scott Raber WDD 130 Final Project Report

I had many opportunities that taught me not just how to make a website, but also the kind of process that goes on to develop one.

As a Client

I had to be creative to come up with content for my site. Elliot also made some extra content on his own, which was great. I got the images from google, but I know in the future I should look specifically for royalty-free images, provide my own, or get a license. It was fun to imagine my site, and then draw and write down the details. This was good practice for planning a website before jumping into writing the code for it.
As the Client, I gained good experience in communicating my wants. I got great feedback by seeing what I got from how I communicated it to the client. Reviewing the page, I could easily see where I had fallen short in communication and what I needed to be more specific about. It was a bit of trial and error, but I ended up very satisfied with the end result.

As a Developer

As the developer, I came to appreciate more the developer of the website where I was the client. There were many instances where I lacked information on certain details and had to fill in with my own decisions. This also helped me realize what sorts of things I could have missed when communicating with my developer. I was able to report my progress and ask for feedback from my client, and make changes so that the site matched his vision.
If I did this project again, I would know better from the start what sorts of things I would need specific details for so that less time is spent asking question after question as they pop up. Instead I'd be better able to get them all at once just by knowing what to look for in a good website plan. I definitely feel more confident in my coding abilities as far as HTML and CSS now that I have done a project from the ground up without a template helping me.
GitHub and GitHub Pages were eaiser to use than I thought they would be. I used GitHub Desktop to get the site on GitHub, and from there it was easy to publish it to Pages.

General Experience and Reflections

Learning the basics of HTML and CSS has been fun and fulfilling. I'll definitely be taking other web design classes in the future. I appreciated the experience of doing a project as though it was a real product being designed for a client. I learned a lot of new ways to use HTML and CSS by designing a website from scratch. It was hard, but it felt really good to get it done.
